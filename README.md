# 🕹️ Hangman Game — 3 languages

Classic word-guessing game (hangman) implemented in **Python**, **JavaScript (browser)**, and **C++ (console)**.

## 🎮 Play / Run

| Language | File | Run command |
|:---|:---|:---|
| Python | `python/hangman.py` | `python hangman.py` |
| JavaScript | `javascript/index.html` | Open in any browser |
| C++ | `cpp/hangman.cpp` | `g++ hangman.cpp -o hangman && ./hangman` |

## 📝 How to play

- The computer chooses a random word.
- You guess letters one by one.
- Each wrong guess draws part of the hangman.
- Guess the word before the hangman is complete!

## 🧠 Features

- 10+ words in the dictionary
- Displays current progress (e.g., `_ _ _ _ _`)
- Tracks wrong guesses
- Simple ASCII graphics (C++ and Python)
